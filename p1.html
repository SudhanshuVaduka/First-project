There are two types of elemnts known as block elments and inline elments

examples of block elemnts are h1,h2,h3...tags and exmaples of inline elements are strong, bold,img tags...etc

The difference is that block elements take up the entire space/width whereas inline elements take only the space required.

Eg:

"This is a block element"

"This is an inline element"

In block elements u can add a inline element but the reverse i.e block element in inline is not possible

This section is about table tag!

Table tag contains the follwoing tags i.e "tr" which stands for 'table row' inside which we have "th" which stands for 'table heading' and "td" which stands for 'table data'

In table tag u can add the attributes like width, for width of the table, height attribute, specifying height of the table and border attribute specifying the border thickness of the table
